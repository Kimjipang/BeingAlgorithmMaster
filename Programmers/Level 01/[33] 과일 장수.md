🧑‍💻 프로그래머스의 [과일 장수] 문제이다. 상자 1개에 m개를 담을 수 있고, k는 최상품 사과 점수이다. 과일 장수가 가능한 많은 사과를 팔았을 때, 얻을 수 있는 최대 이익을 계산하는 것이다.

```python
def solution(k, m, score):
    answer = 0
    score.sort()
    cnt = 0
    for i in range(len(score) - 1, -1, -1):
        cnt += 1
        if cnt % m == 0:
            answer += score[i] * m

    return answer
```
