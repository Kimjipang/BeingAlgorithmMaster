🧑‍💻 프로그래머스의 [푸드 파이터 대회] 문제이다. 푸드 파이터 대회에 쓰일 음식들의 갯수가 담긴 리스트 food를 입력 받는다. 

```python
def solution(food):
    answer = ''
    for i in range(1, len(food)):
        if food[i] >= 2:
            for _ in range(food[i] // 2):
                answer += str(i)
    answer += '0'
    
    for i in range(len(answer) -2, -1, -1):
        answer += answer[i]
        
    return answer
```
