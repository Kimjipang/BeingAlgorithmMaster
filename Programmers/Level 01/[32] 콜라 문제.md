🧑‍💻 프로그래머스의 [콜라 문제]이다. 양의 정수 a, b, N을 입력 받고 빈 병 a개를 줬을 때 새로운 콜라를 b개 준다. 현재 가진 빈 병은 20개이다. 최대로 교환할 수 있는 새로운 콜라가 몇 병인지를 구하면 된다. 



```python
def solution(a, b, n):
    answer = 0
    while n // a > 0:
        received_cola = (n // a) * b
        rest_cola = n % a
        n = received_cola + rest_cola
        answer += received_cola 
        
        
    return answer
```
